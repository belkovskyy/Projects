# Classical Machine Learning

В этом разделе собраны проекты по классическому машинному обучению
(задачи классификации), с фокусом на полный ML-пайплайн:
подготовка данных, обучение моделей, подбор гиперпараметров и оценка качества.

## Проекты

### Прогнозирование оттока клиентов банка
**Файл:** `Прогнозирование оттока клиентов банка на основе машинного обучения.ipynb`

**Задача:**  
Построить модель, прогнозирующую уход клиента из банка, чтобы бизнес мог заранее
принимать меры по удержанию клиентов.

**Что сделано:**
- Загрузка и подготовка табличных данных
- Обработка категориальных и числовых признаков
- Анализ баланса классов
- Обучение моделей без учёта дисбаланса
- Улучшение качества моделей с учётом дисбаланса классов
- Сравнение нескольких алгоритмов
- Финальное тестирование модели

**Используемые модели:**
- Логистическая регрессия
- Дерево решений
- Случайный лес

**Метрики качества:**
- F1-score (основная)
- ROC-AUC (дополнительно)

**Инструменты:**
- `Python`
- `pandas`, `numpy`
- `scikit-learn`
- `matplotlib`, `seaborn`

### Рекомендательная система тарифов мобильной связи
**Файл:** `Рекомендательная система тарифов мобильной связи на основе машинного обучения.ipynb`

**Задача:**  
Построить модель классификации, которая на основе поведения пользователя
рекомендует подходящий тарифный план мобильной связи.

**Что сделано:**
- Разделение данных на обучающую, валидационную и тестовую выборки
- Обучение и сравнение нескольких моделей классификации
- Подбор гиперпараметров
- Оценка качества моделей на валидационной и тестовой выборках
- Проверка модели на адекватность

**Используемые модели:**
- Логистическая регрессия
- Дерево решений
- Случайный лес

**Метрика качества:**
- Accuracy

**Инструменты:**
- `Python`
- `pandas`, `numpy`
- `scikit-learn`

## Навыки, продемонстрированные в разделе
- Классическое машинное обучение (classification)
- Подготовка табличных данных
- Работа с дисбалансом классов
- Подбор и сравнение моделей
- Оценка качества моделей (F1, ROC-AUC, Accuracy)
- Полный ML-пайплайн: от данных до финальной модели
