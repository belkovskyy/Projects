# Statistical Analysis & A/B Testing

В этом разделе представлены проекты, посвящённые статистическому анализу данных и проверке гипотез.
Основной фокус — формулировка бизнес-гипотез, выбор статистических критериев, расчёт метрик и интерпретация результатов с точки зрения бизнеса.

## Проекты

### Анализ тарифных планов компании «Мегалайн»
**Файл:** `Анализ тарифных планов компании «Мегалайн».ipynb`

**Задача:**  
На основе данных о поведении клиентов оператора связи определить, какой тарифный план приносит большую выручку, и проверить соответствующие статистические гипотезы для принятия управленческих решений.

**Что сделано:**
- Загрузка и предобработка данных из нескольких источников
- Приведение временных признаков к корректному формату
- Агрегация данных по пользователям и месяцам
- Расчёт помесячной выручки с учётом условий тарифов
- Исследовательский анализ распределений ключевых показателей
- Формулировка нулевых и альтернативных гипотез
- Проверка гипотез с использованием статистических тестов
- Интерпретация результатов с точки зрения бизнеса

**Проверяемые гипотезы:**
- Средняя выручка пользователей тарифов «Смарт» и «Ультра» различается
- Средняя выручка пользователей из Москвы отличается от выручки пользователей из других регионов

**Инструменты и методы:**
- `Python`
- `pandas`, `numpy`
- `matplotlib`, `seaborn`
- `scipy.stats`
- A/B-тестирование
- p-value, уровень значимости (alpha)
