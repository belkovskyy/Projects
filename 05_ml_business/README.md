# Machine Learning for Business

В этом разделе представлены проекты по машинному обучению с выраженным
бизнес-контекстом: оценка рисков, прибыли и принятие управленческих решений
на основе предсказаний моделей.

Основной фокус — не только качество модели, но и интерпретация результатов
с точки зрения бизнеса.

## Проекты

### Выбор региона для бурения скважин: оценка прибыли и рисков
**Файл:** `Выбор региона для бурения скважин, оценка прибыли и рисков (Bootstrap).ipynb`

**Задача:**  
Определить регион для бурения нефтяных скважин, который обеспечит
максимальную ожидаемую прибыль при допустимом уровне риска.

**Бизнес-контекст:**
- Ограниченный бюджет на разработку скважин
- Необходимость учитывать риск убытков
- Решение принимается на основе прогнозов и вероятностных оценок

**Что сделано:**
- Подготовка и анализ данных геологоразведки
- Обучение модели для предсказания объёма запасов сырья
- Оценка качества модели на валидационной выборке
- Расчёт прибыли по выбранным скважинам
- Оценка рисков и доверительных интервалов методом Bootstrap
- Сравнение регионов по ожидаемой прибыли и риску убытков
- Обоснование финального бизнес-решения

**Используемая модель:**
- Линейная регрессия

**Метрики и методы:**
- RMSE
- Bootstrap
- Доверительные интервалы
- Оценка вероятности убытков

**Инструменты:**
- `Python`
- `pandas`, `numpy`
- `scikit-learn`
- `matplotlib`

## Навыки, продемонстрированные в разделе
- Машинное обучение в бизнес-задачах
- Оценка финансовых рисков
- Работа с неопределённостью и вероятностными методами
- Bootstrap и доверительные интервалы
- Принятие решений на основе данных
- Интерпретация ML-результатов для бизнеса
